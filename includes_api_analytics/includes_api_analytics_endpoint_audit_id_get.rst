The ``GET`` method is used to get details for the specified audit.

**Request**

.. code-block:: xml

   GET /organizations/NAME/audits/ID
   
**Response**

The response is similar to:

.. code-block:: javascript

   {
     
   }

**Response Codes**

.. list-table::
   :widths: 200 300
   :header-rows: 1

   * - Response Code
     - Description
   * - ``200``
     - |response code 200 ok|
