A cookbook generated with`chef generate cookbook chefdocs` creates a cookbook named `chefdocs` with the directory structure:

```text
chefdocs
 ├ .delivery
 | └ project.toml
 ├ recipes
 | └ default.rb
 ├ test
 | └ integration
 | | └ default
 | | | └ default_test.rb
 ├ .gitignore
 ├ CHANGELOG.md
 ├ LICENSE
 ├ Policyfile.rb
 ├ README.md
 ├ chefignore
 ├ kitchen.yml
 └ metadata.rb
```
